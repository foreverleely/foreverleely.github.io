AFNetworking 3.x 和AppStore审核网络要求

2016-08-10

2016 06 01 后，苹果对应用进行审核时会使用ipv6网络，针对传统的ipv4就会因翻译协议不同而出现，审核人员在ipv6环境下无法访问应用内服务网址。
解决方案：对于大多数应用网络访问基本使用AFN，我们只需要使用3.x版本的AFN即可解决问题。注意3.x后，AFN不在支持AFHTTPRequestOperation和AFHTTPRequestOperationManager，这里改用AFHTTPSessionManager即可（当然，如果你使用的是域名，而不是IP，就不存在上述这些问题了）。

结合Xcode进行工程项目管理

2016-07-29

1.假如新建了本地项目，想要放到github上，首先，去github创建远程仓库地址，注意设置好忽略文件以及忽略文件的语言类型
2.使用终端
第一步 切换到项目目录下（注意目录层级）执行 git init
第二步 git remote add origin 远程仓库地址
第三步 git pull origin master (如果上一步使用SSH密钥，那这一步会要求填SSH密钥密码)（有时需要用到vim命令，连按两次大写的z即可退出）
3.打开Xcode，在source control 打开working coples下的configure，我们就会发现本地仓库地址和远程仓库地址都有了
4.直接使用Xcode就可以进行代码管理了
5.重中之重，切记，别项目创建了很多文件后再commit push，不然你会巨后悔.
6.注意需要提交的文件，目录分为三块
01.使用cocoapod生成的xcworkspace；
02.原项目的xcodeproj；
03.pods；
第一部分不用提交；第二部分出项目文件提交外，注意还要提交工程文件及其设置文件，第二部分下的第一个就是，其他userdata不需要提交；第三部分只需提交Podfile文件;
