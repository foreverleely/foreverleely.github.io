Hexo安装使用心得

1.Mac上安装nvm以及node
个人觉得相对好用的方法

git clone https://github.com/creationix/nvm.git ~/.nvm

然后

cd ~/.nvm

执行

export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh" # This     loads nvm

nvm就安装成功了

使用以下可以查看nvm可以安装的node.js的版本

nvm ls-remote

安装指定版本node

nvm install v4.4.0

查看当前安装的node版本

node -v

查看当前使用的node版本

nvm ls

2.安装hexo
首先

npm install -g hexo

然后新建一个文件夹，在该文件夹建立博客所需要的所有文件
cd到那个目录下

hexo init
npm install

初始化和发布本地hexo

exo generate//可以简写为 hexo g
hexo server//可以简写为 hexo s

就可以在浏览器中输入localhost:4000查看本地博客了。要注意的是，所有的hexo指令都必须在你所建立的目录下执行。

3.部署到Github上

将你的本地博客deploy到GitHub上就好。输入以下指令：

npm install hexo-deployer-git --save

然后打开博客目录的_config.yml文件，设置deploy的type为git，repository(repo)
为：https://github.com/你的id/你的id.github.io.git（切记注意需要在type和repo后面加上一个空格再填写，repo和branch是自己添加进去的）

￼￼￼。最后在你的Hexo目录中找到source文件夹，在其中新建一个文件CNAME，没有后缀名（建议使用Sublime Text新建），内容为你的域名（即ＸXXXXX．XXXXXX，不要加ＷＷＷ之类的前缀），保存。输入以下指令：

hexo g
hexo deploy//可简写为 hexo d

在浏览器中输入你自己的域名就可以访问你自己的博客啦！
Github静态网页地址
https://github_id.github.io

4.博客更新
在步骤2中新建的文件夹下

这里需要注意一下，现在需要先启动nvm才能接下去操作

source ~/.nvm/nvm.sh

然后执行

hero new “文件title”

之后在当前目录下的source/_posts/下，可以找到自己新建的那个.md文件；

使用编辑器打开就可以进行编辑，当前自己使用的是Mou
